Function Defintion

a Javascript function is Defined or declared using the
function keyword,following by:

  • The name of the Function.
  • A list of parameters(optional)enclosed in parantheses().
    multiple pramameters are separated by commas(,).
  • The block of codes or the statements enclosed in curly braces{}.

syntax

// codes or statements to be executed.

}

Parameters act as Placeholder variables inside the function.

when the function is called, the variables are assigned to the argument(values) provided when
the function is called.

Js Objects

What is an Object?

object is a collection of related data and or functionality.
creating object an object begins with defining a variable.

var person={
fullName: "Pandeeswaran",
age: 30,
hobbies:["coding" , "read books","listen song"],
isEducated:true
}

you see above syntax of an object is very simple.
The person object contains properties and properties are separated by commas(,).
the name and value pair are separated by a colon(:).

Accessing Object's Properties

two ways to acces an object membeers(properties and methods).

  • The Dot Notation
  • The Bracket Notation

The Dot Notation

The Dot Notation use a dot(.)to access an objects members.
First write the variable,then the name of a property or method.

person.name
person.age
person.hobbies[0]
